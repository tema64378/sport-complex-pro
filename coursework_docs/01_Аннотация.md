# Аннотация

Курсовой проект посвящен разработке локальной информационной системы управления спортивным комплексом.

В рамках работы рассмотрены теоретические основы проектирования информационных систем для предметной области фитнес- и спортивных услуг, выполнен анализ требований, обоснован выбор технологического стека и разработана архитектура программного комплекса.

Программная реализация выполнена в виде клиент-серверного веб-приложения. Клиентская часть реализована на React (Vite), серверная часть — на Rust (Axum), хранение данных организовано в SQLite. Система обеспечивает базовые функции управления пользователями, клиентами, расписанием, бронированиями, платежами и отчетностью.

Практическая часть включает описание этапов разработки, структуры проекта, модели данных, API-взаимодействия, сценариев тестирования и руководств по эксплуатации.

Результатом курсового проекта является работоспособная локальная система, комплект проектной документации и материалы для защиты.

Ключевые слова: информационная система, спортивный комплекс, Rust, React, SQLite, Axum, автоматизация, веб-приложение, UML, IDEF.
