# Чеклист скриншотов для приложения

Для защиты рекомендуется подготовить и вставить в ПЗ/презентацию следующие скриншоты:

1. Экран авторизации.
2. Главная панель (dashboard).
3. Раздел «Клиенты» (таблица + форма создания).
4. Раздел «Тренеры».
5. Раздел «Тренировки/Календарь».
6. Раздел «Бронирования».
7. Раздел «Платежи».
8. Раздел «Отчеты».
9. Раздел «База данных / администрирование».
10. Пример ролевого ограничения (доступ запрещен).
11. Пример успешного сохранения данных.
12. Пример валидации (ошибка ввода).

## Требования к скриншотам

1. На каждом скриншоте должен быть виден заголовок раздела.
2. Разрешение не ниже 1280x720.
3. Единый стиль подписей: «Рисунок N — ...».
4. Нумерация сквозная по ПЗ.
