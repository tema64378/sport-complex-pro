# coursework_docs

Полный комплект документов для курсового проекта по дисциплине ТРПО.

## Порядок использования

1. `01_Аннотация.md`
2. `02_План_график.md`
3. `03_Пояснительная_записка.md`
4. Приложения `04`–`13`

## Быстрая проверка комплектности

- Аннотация: есть
- План-график: есть
- Пояснительная записка: есть
- UML/IDEF/архитектура/ER: есть
- Реальные SVG-диаграммы: есть (`diagrams/*.svg`)
- SQL-схема БД: есть (`16_SQL_схема_БД.sql`)
- Тест-кейсы: есть
- Руководства: есть
- Чеклист скриншотов: есть
- Структура презентации: есть

## Примечание

Перед финальной сдачей:

1. Добавить реальные скриншоты в ПЗ.
2. Проверить требования кафедры к титулу и нумерации.
3. Сформировать финальный PDF с итоговой версткой в Word.
